# 📁 File & Folder Operations

## 🔹 What are File Operations?

File operations are used to create, delete, copy, and manage files/folders.

---

## 🔹 mkdir (Make Directory)

👉 Create a new folder

```bash
mkdir my-folder
```

## 🔹 touch

👉 Create a new file

```bash 
touch file.txt
```

## 🔹 rm (Remove File)

👉 Delete a file

```bash
rm file.txt
```

## 🔹 rm -r (Remove Folder)

👉 Delete a folder

```bash 
rm -r my-folder
```

## 🔹 cp (Copy)

👉 Copy files

```bash 
cp file.txt copy.txt
```

## 🔹 mv (Move / Rename)

👉 Move or rename files

```bash 
mv file.txt newname.txt
```

👉 Move file to another folder:

```bash 
mv file.txt folder/
```

## 🧠 Real Use Case
	•	Create project → mkdir
	•	Create file → touch
	•	Delete unwanted file → rm
	•	Rename file → mv

⸻

## 🚀 Summary
	•	mkdir → create folder
	•	touch → create file
	•	rm → delete file
	•	cp → copy file
	•	mv → move/rename file
