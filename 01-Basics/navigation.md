# 🧭 Terminal Navigation Commands

## 🔹 What is Navigation?

Navigation means moving between folders (directories) in the terminal.

---

## 🔹 pwd (Print Working Directory)

👉 Shows current location

```bash
pwd
```

## 🔹 ls (List Files)
👉 Shows files and folders in current directory

```bash
ls
```

## 🔥 Advanced:

```bash
ls -la   # show hidden files + details
```

## 🔹 cd (Change Directory)
👉 Move into a folder

```bash
cd folder-name
```

## 🔹 Go back:

```bash 
cd ..
```

## 🔹 Go to home:
```bash 
cd ~
```

## 🔹 clear
👉 Clears terminal screen

```bash 
clear
```

## 🧠 Real Use Case
	•	Check where you are → pwd
	•	See files → ls
	•	Move into folder → cd

⸻

## 🚀 Summary
	•	pwd → current location
	•	ls → list files
	•	cd → move between folders
	•	clear → clean screen
